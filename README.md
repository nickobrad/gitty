## Description
This is a website that allows you to search for GitHub Users and GitHub repositories.
### Prerequisites
All you need is a browser installed on your machine.
### Setup
To access this project on your local files, you can clone it using these steps
1. Open your terminal
2. Use this command to clone `$ git clone https://github.com/nickobrad/gitty
3. This will clone the repositoty into your local folder
4. Thank you.
### BDD
1. Search for a user.
2. See a user's repositories
